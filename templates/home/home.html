Restaurant revenue is a modern demo web-application. First of all, it is highly suitable for restaurant owners, but also can be useful for consulting agencies. The application helps to predict restaurant revenue.

The analysis requires only the following initial data:

  • Restaurant opening date;
  • Type of the city: big or not;
  • Type of the restaurant: FC: food court, IL: inline, DT: drive thru, MB: mobile;
  • Demographic data include population in the area, age and gender of the target audience, development scales;
  • Real estate data mainly related to the front facade, parking, etc.
  • Commercial data include the existence of points of interest like schools, banks and other QSR operators.

Restaurant revenue works on the basis of a neural network developed on the Random Forest architecture. The algorithm is guaranteed to be accurate, as the dataset used to train the model uses data from 137 restaurants.

Here are some of key features:

  • Lightweight and clean design
  • User-friendly interface
  • Requires only 10-15 seconds to get the result you need
  • This service has the ability to train a model on its own dataset
  • All input datasets must be in the same format
  • .cvs format of income files
